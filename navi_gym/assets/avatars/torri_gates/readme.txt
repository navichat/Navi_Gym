MikuMikuDance用データ「千本鳥居風ステージ」

【はじめに】
ダウンロードしていただき有難うございます。
このステージは、頂点数多目ですが、尚且つ全体が草や葉っぱ(板ポリのアルファ抜き)だらけで、
その影を良く見せたい時にはMMEが必要かと。以下草・葉っぱの影の事を少々。


●使い方の一例
どちらかの千本鳥居風ステージ(pmx)をMMDに読み込んでください。セルフ影をモード2にしてみる。
全て板ポリである草・葉っぱの影は当然、四角ばっています。
なので、テクスチャの透明部分を抜いた影にするために、
例えば、データP様のDetailUp_for_Landscape_v055(http://ux.getuploader.com/Elle_DataPort/index)を使ってみます。

・64bit版MMD/MMEの場合は、Model_on_Landscape.fxもしくはAdultS2_on_Landscape.fxをテキストエディタで開き、
　//#define MIPMAP_MMD64 // 64bit版MMD/MMEの機能を有効にする
　という行の行頭の//を取る。
　
・DetailUp_for_Landscape_v055内のShadowMap.xをMMDに読み込む。

・[MMEffect]メニューから、[エフェクト割当]を選択し、エフェクト割当ダイアログを表示。
　[Main]タブで、千本鳥居風ステージの欄を選択し、Model_on_Landscape.fxもしくはAdultS2_on_Landscape.fxのどちらかをドラッグ&ドロップ。

・セルフ影のモード2のスライダを動かして、なるべく影のギザギザが出ないようにする、以上。

・またそぼろ様のAutoLuminousを読み込めば、灯篭が発光します。

・消失点明度調整モーフで、周囲と消失点画像の明るさを調整できます。
